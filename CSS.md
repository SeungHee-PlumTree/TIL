#CSS
---
- `<style></style>` : Now CSS's time to work

   - example
   ```
   <style>
   이 웹페이지에 있는 모든 li 태그의 
   폰트 컬러를 붉은색으로 칠해주세요.
   </style>
   ```
   **--->**
   ```
   <style>
   li{
    color: red;
   }
   </style>
   ```
- `text decoration : #;`
    - `underline` : 글자 밑줄
    - `line-through` : 글자 중간에 줄
- `border:#px` : 테두리 두께
- `solid #;` : 테두리 색깔
- `background-color: #;` : 배경색
- `a:#`
  - `hover` : 마우스 올렸을때
  - `active` : 마우스 눌렸을때
---
##선택자
- `id = "해파리"` : 해당 id(개인)만 디자인 
   - style 안에 `#해파리{}` 넣기
- `class = "해파리"` : 해당 class(그룹)만 디자인 
   - style 안에 `#해파리{}` 넣기
- `#부모>자식` : 부모>자식 에 디자인
